# OG Game Challenge

## Overview

OG Game Challenge is a web application that brings together a collection of retro-inspired mini-games within a modern and interactive platform.

The project combines a Python backend, a responsive frontend, and JavaScript-based games connected through a REST API. Players can create a profile, play different challenges, submit their scores, and compete through online leaderboards.

Designed as a collaborative software engineering project, the application focuses on clean architecture, modularity, and maintainability while providing an engaging gaming experience.

---

## Main Features

- User registration
- Multiple arcade mini-games
- Score tracking
- Online leaderboards
- Responsive web interface
- RESTful API communication
- Persistent database storage

## Project Goals

The objective of OG Game Challenge is to demonstrate the design and implementation of a complete full-stack application by integrating:

- a modular backend architecture,
- an interactive frontend,
- browser-based games,
- persistent score management,
- and a scalable software structure suitable for future expansion.
